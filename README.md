derby-standalone
================

Client-side only version of Derby


# Usage

## Hosted
```
<script src="http://codeparty.github.io/derby-standalone/dist/0.6.0-alpha25/derby-standalone.min.js"></script>
```

## Bower
```
bower install derby-standalone
```
