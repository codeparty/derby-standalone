derby-standalone
================

Client-side only version of [DerbyJS](http://derbyjs.com/)


# Usage

## Hosted
```
<script src="http://derbyjs.github.io/derby-standalone/dist/0.6.0-alpha60/derby-standalone.min.js"></script>
```

## Bower
```
bower install derby-standalone
```
